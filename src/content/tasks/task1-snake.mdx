## Task Brief

The agent controls a snake in a bounded grid. Each action changes movement direction, and the score rewards sustained food collection without crashing.

## Why It Matters

Snake tests whether a model can combine visual state parsing, short-horizon route planning, and rule consistency over several steps.

## Author Note

A familiar arcade rule set becomes a useful probe for planning under partial future risk.
