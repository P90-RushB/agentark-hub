## Task Brief

Extract appointment details from a mock message thread and create the matching calendar event.

## Why It Matters

Calendar creation from messages measures whether a model can parse visible communication, keep the relevant fields in memory, and produce the right structured GUI result.

## Author Note

The task evaluates cross-app transfer: read details in one app, retain them, and enter the correct event fields in another.
